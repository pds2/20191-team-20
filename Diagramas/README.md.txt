ExKalibur 1.0

===========================================================================

1.REQUISITOS
----------------------------

 O único requisito é um compilador de código em C++.


2.INSTALAÇÃO E CONFIGURAÇÃO
----------------------------

 O jogo foi desenvolvido em linguagem C++ e roda em terminal. 
 Ainda não foi desenvolvido um arquivo executável para o jogo, 
 portando ainda não há como instalar na máquina, somente compilar
 no terminal.

3.COMO JOGAR?
----------------------------

 O jogo é para dois jogadores que são cadastrados na tela inicial.
 Inicialmente o primeiro jogador deve escolher uma carta de movimento
 e posteriormente ataque	entre o carteado que receber com intuito 
 de 'ferir' o adversário. Em seguida o segundo jogador deve escolher uma 
 carta de defesa, uma de movimento e uma de ataque, para cada carta que
 não tiver ou que escolher não lançar o jogador deve colher no monte do
 baralho. Assim o jogo continua até que todas o 'sangue' do adversário
 acabe.
